VSCode Error Detector es una extensión desarrollada para Visual Studio Code que realiza análisis estático de código en tiempo real para identificar errores comunes, malas prácticas y patrones potencialmente problemáticos.

La extensión analiza el Abstract Syntax Tree (AST) del código fuente para detectar inconsistencias, duplicación de lógica o estructuras incorrectas, mostrando advertencias directamente en el editor e integrándose con el panel de problemas.

El proyecto implementa desarrollo modular utilizando la API oficial de VSCode, tipado estricto con TypeScript y arquitectura extensible para definir reglas personalizadas. Representa una herramienta enfocada en mejorar la calidad del software y fomentar buenas prácticas de programación.
