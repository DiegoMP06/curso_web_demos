Kanban Project Manager es una aplicación colaborativa inspirada en herramientas de gestión visual como Trello, diseñada para organizar proyectos mediante tableros dinámicos estructurados en columnas. Permite crear proyectos, administrar tareas, asignar responsables y dar seguimiento al progreso en tiempo real.

El sistema integra comunicación mediante WebSockets utilizando Socket.io, lo que permite sincronización instantánea entre múltiples usuarios conectados. Incluye comentarios, notificaciones y relaciones complejas entre entidades como usuarios, proyectos y tareas.

Técnicamente, el proyecto implementa arquitectura modular, separación clara de responsabilidades y diseño relacional optimizado en PostgreSQL. Representa una solución moderna para colaboración en entornos digitales con enfoque en escalabilidad y rendimiento.
