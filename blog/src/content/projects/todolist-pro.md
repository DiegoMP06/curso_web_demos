ToDoList Pro es una aplicación avanzada de gestión de tareas orientada a mejorar la productividad personal y profesional mediante una experiencia estructurada, dinámica e intuitiva. Permite crear, organizar y priorizar tareas utilizando etiquetas personalizadas, fechas límite, estados dinámicos y filtros inteligentes.

El sistema integra funcionalidad de arrastrar y soltar (Drag & Drop), persistencia en base de datos y autenticación de usuarios, permitiendo mantener información segura y sincronizada. Desde el punto de vista técnico, implementa una arquitectura cliente-servidor desacoplada, manejo avanzado de estado con Redux Toolkit y tipado estricto con TypeScript.

El backend expone una API REST robusta conectada a PostgreSQL, garantizando integridad de datos y escalabilidad. El proyecto demuestra aplicación de buenas prácticas modernas de desarrollo frontend y backend.
